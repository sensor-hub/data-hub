# Data-hub

## Requirements

* Docker
* docker-compose
* Being reachable from the sensor-hub.


## Quickstart

* `docker-compose up`
* go to `http://127.0.0.1:3000/login` with `admin:admin`
* Add a data source : `InfluxDB` with address : `http://influxdb:8089`
* Defaults credentals are : `grafana`, `grafana`
